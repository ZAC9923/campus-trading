# 山大二手工坊

## 项目介绍

基于SSM的校园二手物品交易平台


## 运行环境

开发工具 IDEA

安装运行环境：

1. `jdk1.8` 
2. `Tomact8`
3. `maven3.5`

数据库：MySQL
